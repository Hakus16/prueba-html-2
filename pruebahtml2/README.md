# Burger Festival | Fusión Gourmet

## Description
Fusión Gourmet is a premium gastronomic festival focused on author burgers. This landing page provides information about the event, including schedules, participating chefs, a gallery of our best creations, and booking options.

## Technologies Used
- **HTML5**: Semantic structure for better accessibility and SEO.
- **CSS3**: Advanced styling using Flexbox, CSS Grid, and Media Queries for a responsive experience.
- **Google Fonts**: Custom typography using the "Outfit" font family for a modern and premium look.

## Project Structure
- `prueba.html`: The main entry point of the application containing the semantic HTML structure.
- `style.css`: The stylesheet containing all the visual design, including the responsive media queries.
- `https://fonts.googleapis.com/css2?family=Outfit`: External typography.

## Local Viewing Instructions
1. Clone or download the project files.
2. Open the `prueba.html` file in any modern web browser (Chrome, Firefox, Edge, etc.).
3. To view the responsive version, use the browser's developer tools (F12) and toggle the device toolbar to simulate mobile screens.

## Features
- **Responsive Design**: The site adapts seamlessly to desktop, tablet, and mobile devices.
- **Mobile Gallery**: Specifically optimized for mobile screens with a 3-column grid layout as requested.
- **Premium Aesthetics**: Sophisticated color palette using black, copper, and earth tones.
- **Structured Content**: Information organized with semantic tags like `<header>`, `<main>`, `<section>`, `<footer>`, and a structured `<table>` for schedules.
